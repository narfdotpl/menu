menu
====

Click around OS X menu bar from the command line.

    $ menu Finder Window Minimize
    $ menu Xcode Product Destination 'iPhone Retina (4-inch)'


Installation
------------

Menu is a [shell script](menu), so just add it to your `$PATH`.
